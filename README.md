## 📖 简介
糖果梦热榜是一个实时更新的热门清单，汇集了最新的资讯，涵盖新闻、社交、媒体、GitHub等多个领域。只需打开页面，即可轻松浏览各大热门内容，时刻跟进行业动态，掌握最热话题。

---
## 🌍 时效性
- 数据每分钟更新一次（GitHub20-40分钟，网易云音乐8-15分钟，豆瓣2-10分钟）
- 所有接口均来自官方，无任何第三方中转，主打无情铁手，AA接W接A接无情铁手接外圈刮接一刀斩

---

## 🪠 功能支持

- ✅ 目前支持全网200多个高质量平台，每分钟更新热点1万多条
- ✅ 站内数据分钟级自动更新 (特殊除外: GitHub20-40分钟，网易云音乐10-15分钟，豆瓣2-10分钟，词云1分钟)
- ✅ 自定义选择展示平台 (隐藏自己不想看的平台)（2025-07-07+）
- ✅ 明暗主题切换（2025-07-04+）
- ✅ 支持评论（2025-07-19+）
- ✅ 收藏卡片 (把喜欢的平台单独放到收藏分类下)（2025-11-10+）
- ✅ 分类自定义：名称字体大小、排序、默认分类
- ✅ 卡片自定义：宽度、高度、名称字体大小、标题是否完整显示、更新时间是否显示、卡片拖动排序、移动端横向纵向滚动切换
- ✅ 热点自定义：标题字体大小、标题是否完整显示、热度值是否显示
- ✅ 页面关键词过滤 (包含关键词或者排除关键词)
- ✅ 支持RSS (全站、单分类、单平台 全都支持)
- ✅ 全网实时热点词云 (分钟级更新支持自定义展示隐藏)
- ✅ 全网实时热点检索 (实时从全网数万个热点中检索自己想看的)
- ✅ AI实时热点简报 (针对近0-5分钟内全平台出现的所有热点，通过AI大模型进行总结、分析、预测后出的一份报告，可以实时查看)
- ✅ 支持摸鱼模式 (超级逼真的EXCEL和VSCODE摸鱼模式)
- ✅ 订阅推送(支持飞书、钉钉、企业微信、Telegram、NTFY、GOTIFY)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/CandyDream6/tgmeng-top-search-frontend)

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/CandyDream6/tgmeng-top-search-frontend)

- 设置环境变量 VUE_APP_API_BASE_URL=<你的后端地址>/api/ 例如 http://127.0.0.1:4399/api/

### 2 Docker镜像一键部署

```shell
docker pull tgmeng/tgmeng-top-search-frontend:latest                     # 这是dockerhub里的镜像
# docker pull ghcr.io/tgmeng-com/tgmeng-top-search-frontend:latest       # 这是ghcr里的镜像，和上面是一样的，拉哪个都行
docker run -d -p 8080:80 -e VUE_APP_API_BASE_URL=<你的后端地址>/api/ --name tgmeng-top-search-frontend tgmeng/tgmeng-top-search-frontend:latest
```

- 下载本项目根目录下的docker-compose.yml，然后在他的同级目录执行下面命令

```shell
VUE_APP_API_BASE_URL=<你的后端地址>/api/ docker-compose up -d
# 例如:VUE_APP_API_BASE_URL=http://127.0.0.1:4399/api/ docker-compose up -d
```
